# Intel

```text
Intel-inspired front-end practice site
Built to learn layout, styling, and UI interactions
```

This repository is a front-end learning project where I recreated the look-and-feel and a few interactions inspired by Intel’s public marketing pages.

```text
DISCLAIMER
This is an educational site copy made for learning and portfolio practice only.
Not affiliated with Intel. Not an official Intel website/product.
Intel name, logos, and related marks belong to their respective owners.
```

## What’s inside

```text
- Fixed navigation with dropdown-style panels
- Scroll-driven hero section (slides + active indicator dots)
- Content sections and footer layout
- Canvas-drawn logo
```

## Media formats used (learning goal)

```text
PNG  → Favicon (img/faviconV2.png)
SVG  → Navigation logo (img/intel-header-logo-homepage.svg)
Canvas → Footer logo rendered via JS (canvas#logoCanvas + js/logo.js)
```

## Built with

```text
HTML5
CSS3
Vanilla JavaScript (DOM + events)
Canvas API (logo rendering)
Fonts: Roboto (local)
```
